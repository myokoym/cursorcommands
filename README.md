# Cursor 自作コマンド集

Cursor IDEで使用する自作コマンドを管理するリポジトリです。

## 概要

このリポジトリには、Cursor IDEのカスタムコマンド定義ファイルを格納しています。プロジェクト間で再利用可能なコマンドを集約し、一元管理します。

## 含まれるコマンド

### commit-atomic

原子的コミット実行コマンド。論理的に意味のある最小単位で、タスクに関連するファイルのみをコミットします。

- **ファイル**: `commit-atomic.md`
- **説明**: 最小単位のコミット、個別ファイル指定、論理的な分離を徹底したコミット実行

## 使用方法

Cursor IDEのコマンドパレットから、以下のようにコマンドを実行できます：

```
/commit-atomic [commit-message]
```
