# Cursor 自作コマンド集

このディレクトリには、Cursor IDEで使用する自作コマンドを格納しています。

## 概要

このディレクトリは、将来的にサブモジュール化する予定です。プロジェクト固有のカスタムコマンドを管理するための専用リポジトリとして運用します。

## 含まれるコマンド

### commit-atomic

原子的コミット実行コマンド。論理的に意味のある最小単位で、タスクに関連するファイルのみをコミットします。

- **ファイル**: `commit-atomic.md`
- **説明**: 最小単位のコミット、個別ファイル指定、論理的な分離を徹底したコミット実行

## 使用方法

Cursor IDEのコマンドパレットから、以下のようにコマンドを実行できます：

```
/commit-atomic [commit-message]
```

└── commit-atomic.md    # 原子的コミット実行コマンド
